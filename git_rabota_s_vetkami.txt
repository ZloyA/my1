git branch  - показ в какой ветке мы находимся ( количество веток)
git branch имя ветки - создание новой ветки
git checkout имя ветки - переход на созданную ветку
touch имя файла - создание файла с именем
vim имя файла - редактирование содержимого файла
выход из файла
git add . -  запись в гит ветки
git commit -m "комментарий " 
git checkout -b имя ветки -  создание и заход в ветку 
если надо скопировать из папки в папку - находясь в папке в которую перенести команда git merge имя папки из которой переносим
git push -u origin название папки  - перенос новой папки в гитхаб
